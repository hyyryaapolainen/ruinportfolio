Overview

I helped streamline content by giving feedback if something is currently possible within the systems of our game or if something needs to be custom fitted.


For example, adding channeled skills would've taken a new implementation to the original skill system and as such was put on hold.


I also created the Visual effect for the player slash and placeholder VFX for skills using Unity's particle system and VFX graph.


All of the shaders in game are made in Shader Graph with additional HLSL from Unity's Open Project 1


These experiments helped me understand the workflow from the drawing board to in-game.
